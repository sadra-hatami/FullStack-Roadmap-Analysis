ضروری CSS3 ظاهر و جذابیت وب ▼
چیه؟
زبان استایل‌دهی به HTML. رنگ، فاصله، فونت، انیمیشن، ریسپانسیو — همه با CSS.
Flexbox vs Grid
Flexbox برای چیدمان یه بعدی (ردیف یا ستون). Grid برای چیدمان دو بعدی (ردیف و ستون با هم). هر دو ضروریه.

CSS Variables

ضروری JavaScript (ES6+) روح وب — بدونش نیمه‌ای ▼
چیه؟

ES6+ یعنی چی؟
نسخه‌های مدرن JS از ۲۰۱۵ به بعد. arrow functions، destructuring، async/await، modules — اینا ES6+ هستن.
DOM چیه؟

Async/Await
روش مدیریت عملیات ناهمزمان مثل fetch کردن داده از API. بدونش نمیتونی با backend ارتباط برقرار کنی.

Virtual DOM

Hooks چیه؟
توابع خاص React مثل useState، useEffect، useContext که مدیریت state و side effects رو ممکن میکنن.
✅ مزایا

  • بیشترین job posting

  • Meta پشتشه

چیه؟
زبان برنامه‌نویسی high-level که هم برای backend وب، هم AI/ML، هم automation استفاده میشه.
چرا برای تو بهتره؟
تو قبلاً Python یاد گرفتی. با Django یا FastAPI میتونی backend قوی بسازی. مهاجرت با Python راحت‌تره چون AI هم بلدی.

Python + Django یا FastAPI در اروپا و کانادا تقاضای بالایی داره. ترکیب با AI مهارتت رو دو برابر ارزشمند میکنه.

بعدش چی؟
Django برای اپ‌های کامل، یا FastAPI برای microservices و API-only.
ضروری Django باتری‌دار — همه چیز داخلشه ▼
چیه؟
فریمورک Python برای web development. "batteries included" — authentication، admin panel، ORM، migration همه داخلشه.
Django REST Framework
افزونه Django که ساخت API رو آسون میکنه. برای Fullstack حتماً باید DRF هم یاد بگیری.
کجا استفاده میشه؟
Instagram، Pinterest، Disqus، Mozilla — اینا همه از Django استفاده کردن یا میکنن.
Django vs FastAPI
Django: full-featured، برای اپ‌های کامل. FastAPI: سریع‌تر، مدرن‌تر، برای API-only و microservices.
✅ مزایا
  • همه چیز داخلشه
  • Admin panel رایگان
  • امنیت قوی پیش‌فرض
  • ORM قوی
❌ معایب
  • برای پروژه‌های کوچیک سنگینه
  • نسبتاً کنده‌تر از FastAPI
  • منحنی یادگیری

پیش‌نیاز

🛠
پروژه تمرینی

چرا مهمه؟

ضروری RESTful APIs زبان مشترک Frontend و Backend ▼
چیه؟
استانداردی برای طراحی API. بر اساس HTTP methods (GET, POST, PUT, DELETE) کار میکنه. هر resource یه URL داره.
مثال واقعی
GET /api/users/ — لیست کاربران. POST /api/users/ — کاربر جدید. DELETE /api/users/1/ — حذف کاربر ۱.
Status Codes
200 OK، 201 Created، 400 Bad Request، 401 Unauthorized، 404 Not Found، 500 Server Error — باید همه رو بدونی.
JSON چیه؟
فرمت استاندارد تبادل داده بین frontend و backend. هر API داده رو به شکل JSON برمیگردونه.
✅
چرا یاد بگیری؟
این پل ارتباطی Frontend و Backend هست. بدون REST، Fullstack معنی نداره.
🛠
پروژه تمرینی
با Postman یا Thunder Client، API‌های مختلف (مثل CoinGecko که استفاده کردی) رو test کن و Request/Response رو بررسی کن.
ضروری JWT / OAuth امنیت Authentication ▼
JWT چیه؟
JSON Web Token — روشی برای تایید هویت کاربر. بعد از login، یه token میگیری که با هر request میفرستی.
OAuth چیه؟
پروتکل authorization برای "Login with Google/GitHub". بدونش نمیتونی social login پیاده‌سازی کنی.
✅
چرا یاد بگیری؟
هر اپ واقعی Authentication نیاز داره. این یه مهارت اساسیه که در همه پروژه‌ها استفاده میشه.

ضروری PostgreSQL بهترین Relational Database ▼
چیه؟
قوی‌ترین open-source relational database. داده رو در جداول با ارتباط بین هم ذخیره میکنه. از SQL استفاده میکنه.
چرا PostgreSQL نه MySQL؟
PostgreSQL feature-rich‌تره، JSON داخلی داره، advanced indexing داره، و برای production-grade app‌ها استانداردتره.
SQL یعنی چی؟
Structured Query Language — زبان کوئری زدن به database. SELECT، INSERT، UPDATE، DELETE — این‌ها دستورات پایه SQL هستن.
کجا استفاده میشه؟
Instagram، Spotify، Reddit، و اکثر شرکت‌های بزرگ. Django هم PostgreSQL رو به عنوان primary database توصیه میکنه.
📚
پیش‌نیاز
مفاهیم پایه database — table، row، column، primary key، foreign key، و SQL اولیه.
🛠
پروژه تمرینی
Schema کامل برای سیستم هتل‌داریت طراحی کن — room، booking، user، payment. با SQL کوئری بزن.
توصیه‌شده MongoDB NoSQL — بدون schema ثابت ▼
چیه؟
NoSQL database که داده رو به شکل JSON-like documents ذخیره میکنه. نیازی به schema ثابت نداره.
کِی استفاده کنی؟
وقتی داده‌ات ساختار متغیر داره، یا real-time app میسازی، یا با Node.js کار میکنی.
💼
بازار کار
اگه Node.js stack استفاده میکنی MongoDB خیلی رایجه. ولی برای Python stack، PostgreSQL اولویت داره.
توصیه‌شده Redis سرعت — همه چیز در RAM ▼
چیه؟
In-memory database — داده رو توی RAM نگه میداره. ۱۰۰ برابر سریع‌تر از دیسک. برای caching استفاده میشه.
کجا استفاده میشه؟
Session storage، API caching، rate limiting، real-time features. Twitter از Redis برای timeline استفاده میکنه.
✅
چرا یاد بگیری؟
وقتی سایتت traffic زیاد داشته باشه، بدون caching database کند میشه. Redis این مشکل رو حل میکنه.

غایب — ضروری Git & GitHub بدون این استخدام نمیشی ▼
Git چیه؟
سیستم کنترل نسخه. هر تغییری که به کدت میدی ذخیره میشه. میتونی به هر نسخه قبلی برگردی.
GitHub چیه؟
پلتفرم آنلاین برای Git. رزومه برنامه‌نویسه. کارفرماها اول GitHub تو رو چک میکنن.
دستورات مهم
git init، git add، git commit، git push، git pull، git branch، git merge — اینا پایه‌ هستن.
چرا این قدر مهمه؟
هیچ شرکتی کد بدون Git نمینویسه. اگه Git بلد نباشی حتی نمیتونی اول روز کاریت شروع کنی.
✅
چرا یاد بگیری؟
این نه یه مهارت، بلکه یه requirement هست. بدون Git نمیتونی در هیچ تیمی کار کنی.
❌
اگه skip کنی چی میشه؟
